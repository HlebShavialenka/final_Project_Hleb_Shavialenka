Installing
----------

Install and update using `pip`_:

.. code-block:: text

    $ pip3 install -r requirements.txt

.. _pip: https://pip.pypa.io/en/stable/getting-started/


Create DB
----------------


.. code-block:: text

    $ python3 db_test.py


Run Web
----------------


.. code-block:: text

    $ python3 app.py
      * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
